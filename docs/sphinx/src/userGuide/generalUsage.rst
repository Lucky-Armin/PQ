.. _generalUsage:

#############
General Usage
#############

The PQ applications is based on a single input file execution. A fully prepared setup is completely defined by an input file `PQ.in` and can be executed in the following form:

.. code-block:: bash

    PQ PQ.in

For details on the general file structure of an input file and additional setup files please refer to sections :ref:`inputFile` and :ref:`setupFiles`.
